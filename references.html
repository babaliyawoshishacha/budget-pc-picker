References

This page documents the supporting sources, datasets, and tools used to build this prototype, and acknowledges third-party contributions that informed the design and implementation.

Data Sources

  • Bilibili creator data (芝士华莱士) – Used as a benchmark inspiration for indicative price references and component-market context. This project uses the information for learning and interface prototyping purposes only.
  • Self-curated teaching dataset – A simplified, non-commercial dataset created for the INST0007 prototype demonstration. It does not represent real-time market scraping.

Tools & Assistance

  • ChatGPT – Supported the early-stage layout planning, helped refine text for a consistent professional tone, and assisted with debugging CSS/JS behaviours (e.g., layout alignment, reveal timing, and responsive adjustments). The final structure, decisions, and integration were reviewed and implemented by the author.

Academic Context

This website is a coursework prototype for INST0007. All data and interface elements are used in an educational context. The design prioritises clarity, usability, and responsible interpretation of indicative pricing rather than claiming market accuracy.

Notes on Responsible Use

The budget and used-price outputs are intended to reduce decision friction and demonstrate web interaction patterns. Users should verify current pricing, compatibility, and regional availability through official retailers or trusted marketplaces before making purchase decisions.
